# Vibe Coding - Project Knowledge

## Project Overview
Vibe Coding landing page built with Next.js 14, TypeScript, and Tailwind CSS. Features a modern design with animated components and responsive layout.

## Tech Stack
- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom component library in `src/components/ui/`
- **Animations**: Custom animations and effects

## Project Structure
- `src/app/` - Next.js app router pages
- `src/components/landing/` - Landing page specific components
- `src/components/ui/` - Reusable UI components
- `public/` - Static assets (logos, images, avatars)

## Key Features
- Responsive hero section with animations
- Testimonials with avatar images
- Method steps with local image assets
- Gallery slideshow
- Partner logos
- FAQ section

## Development
- Uses standard Next.js development workflow
- Tailwind for styling with custom configuration
- Component-based architecture
- TypeScript for type safety

## Assets
- Logos stored in `public/logos/`
- Step images in `public/etapes/`
- Avatar images in `public/avatars/`
- Gallery images in `public/slideshow/`
